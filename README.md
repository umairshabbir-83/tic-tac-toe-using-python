# tic-tac-toe-using-python
This repository contains `Tic-Tac-Toe` game implementation files in `Python`.

## Instructions:
You can play â–¶ this game with the use of `NUMERIC PAD`. The game field mapping with respect to `NUMERIC PAD` is shown below.

```
Game Field

		|		|	
----------------+---------------+---------------
		|		|	
----------------+---------------+---------------
		|		|		

Key Field

 	7	|	8	|	9
----------------+---------------+---------------
	4	|	5	|	6
----------------+---------------+---------------
	1	|	2	|	3
```

The entered key will mark your move ("X" or "O") at corresponding position.
